Cypress.io Evaluation
=====================

### Install
```
git clone git@github.com:moarwick/cypress-tests.git
cd cypress-tests
npm i
```

### Run Cypress Dashboard
```
npm run cypress
```
* Cypress app opens
* Make sure your `dev.tradesy.com` is running
* Click `> Run All Tests` (or specific test in the list)
